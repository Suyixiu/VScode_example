# sample_normal
很简单的一个小C++代码。只要你g++的shell命令没错然后-I了你的头文件然后也括进了你的各个cpp那你就可以成功编译运行并调试了  
此外如果你不在c_cpp_properties.json中添加你的头文件的路径呢你会看到烦人的绿色波浪线，所以我老老实实就加了
